Arabic Translation Project

What's Arabic GD?

It's a TP powered by Texture Workshop (mod in geode-sdk) that makes the game translated to Arabic.

Why is Arabic GD suitable for Arabs?

Arabs who can't understand English completely can understand the game after installing the TP.

Note:

This TP is incomplete and some texts are untranslated.

How to install the TP?

  1. Download the TP by pressing the button below:
Download Arabic TP View Source Code
  1. Go to "download" folder and copy the TP file
  2. Then go to geode files and open "config" folder
  3. Go to "geode.texture-loader" folder (if you don't find it, create new one but make sure that you installed "Texture Loader")
  4. Go to packs folder (if you don't find it, create new one) and paste the TP
  5. Open the game and go to Texture Loader to Enable the pack

.. and finally you have Arabic GD
